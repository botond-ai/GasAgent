# Szoftver Telepítési Szabályzat

## Engedélyezett Szoftverek

A következő szoftvereket telepítheted önállóan:

### Irodai Alkalmazások

- Microsoft Office 365 (automatikusan telepítve)
- Google Chrome böngésző
- Mozilla Firefox
- Adobe Acrobat Reader

### Fejlesztői Eszközök

- Visual Studio Code
- Git
- Python 3.x
- Node.js
- Docker Desktop (kérelmezés után)

### Kommunikációs Eszközök

- Microsoft Teams (előtelepítve)
- Slack
- Zoom

## Telepítési Folyamat

### Self-Service Portal

1. Lépj be a https://apps.company.com oldalra
2. Böngéssz a katalógusban
3. Kattints a "Telepítés" gombra
4. Várd meg az automatikus telepítést (5-10 perc)

### Kérvényezés

Ha olyan szoftvert szeretnél, ami nincs a katalógusban:

1. Írj emailt a software@company.com címre
2. Indokold a használat célját
3. Add meg a szoftver pontos nevét és verzióját
4. Várható válaszidő: 2-3 munkanap

### Tiltott Szoftverek

Az alábbi kategóriák telepítése TILOS:

- Peer-to-peer (P2P) alkalmazások
- Torrents
- Nem hivatalos hacking tools
- Vírus/malware eszközök
- Nem licenszelt szoftverek

## Biztonsági Szabályok

- Csak megbízható forrásból tölts le szoftvert
- Mindig a legfrissebb verziót használd
- Rendszeresen frissítsd az alkalmazásokat
- Ha gyanús aktivitást észlelsz, azonnal jelentsd a security@company.com címen
