================================================================================
ARCHITECTURE DIAGRAM
================================================================================

User Request
    ↓
[plan_node]
  - Parse input
  - Search RAG
  - Call LLM
  - Generate plan
    ↓
[executor_loop]
  - For each step:
    ├─→ [tool_router]
    │     - Identify tool
    │       ↓
    │   [action_node]
    │     - Execute tool
    │       ↓
    │   [observation_node]
    │     - Record results
    │     - Update state
    │     - Next step
    │
    └─→ [summary_node]
          - Generate summary
            ↓
        [final_answer_node]
          - Format report
            ↓
        Final Report Output
