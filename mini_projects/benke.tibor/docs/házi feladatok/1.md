Működő API Endpoint (Példa)

POST /api/query/ — Feldolgozz egy HR / IT / Finance / etc. kérdést
curl -X POST http://localhost:8001/api/query/ \
  -H "Content-Type: application/json" \
  -d '{
    "user_id": "emp_001",
    "query": "Szeretnék szabadságot igényelni október 3-4-re"
  }'

  "success": true,
  "data": {
    "domain": "hr",

        "doc_id": "HR-POL-001",
